About

KEEP IT CLEAN

As a developer, I believe in clean and concise code. Maintaining readability for others, as well as my future self, is important to me. Viewing beautifully written code is a pleasure, so I seek to write beautiful code.

KEEP IT SMART

If clean code is important, efficient code is paramount. As new features and technology demand more from our machines, it is important to write code that can help mitigate the strain. Great code gets the job done as quickly and effortlessly as possible. Effecient code also raises the beauty factor, so it's really a win win.

KEEP IT FUN

Teams work best when they can communicate. Teams communicate best when they're having fun. I seek to maintain a positive attitude in every moment, focusing on solutions isntead of problems. Being a useful member of a great team is very rewarding to me.
